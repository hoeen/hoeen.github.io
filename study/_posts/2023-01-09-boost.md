## Cloud
기존은 IDC (Internet Data Center)과 같은 물리적인 공간을 만들고 운영.  

Cloud 서비스는 공간적인 제약에서 벗어나고, 개발자가 직접 설정하는 작업 등도 클라우드에서 쉽게 할 수 있게 Managed 서비스함.

클라우드 서비스의 다양한 제품
- 컴퓨팅 서비스
    - 가장 많이 사용됨. 연산을 수행하는 서비스
- Stateless Container
    - Docker container 기반으로  서버를 실행
- Object Storage
    - 저장소 역할 수행
- Database
    - PostgreSQL 등
- Data Warehouse
    - 데이터 분석에 특화됨.
- AI Platform
